# **App Name**: Crypto Signal Stream

## Core Features:

- Signal Generation: Gere um sinal de negociação de criptomoedas com par, direção, preço, confiança, TP, SL e timestamp. Usa uma ferramenta para calcular valores razoáveis para take profit (TP) e stop loss (SL).
- Signal Streaming: Exiba os sinais gerados em um feed legível e em tempo real.
- Pair Filtering: Permite que os usuários filtrem os sinais exibidos com base no par de negociação de criptomoedas.

## Style Guidelines:

- Cor primária: Verde escuro (#006400) para transmitir confiança e estabilidade.
- Cor secundária: Preto (#000000) para um visual moderno e minimalista.
- Cor de destaque: Verde neon (#39FF14) para destacar informações importantes e elementos interativos.
- Fonte: 'Roboto' (sans-serif) para uma interface limpa, moderna e legível.
- Use ícones simples e baseados em linhas para representar detalhes do sinal (direção, par, etc.).
- Use um layout claro e colunar para apresentar as informações do sinal de forma eficiente.
- Implemente animações sutis para sinais recebidos para chamar a atenção sem ser intrusivo.
